# College Model Website

## Overview
The College Model Website is a responsive web application developed using HTML, CSS, and Bootstrap 5. It presents a modern landing page for a college with a clean and user-friendly design. The project currently consists of a single-page website and is planned to be expanded with additional pages and features in future updates.

## Features

- Responsive Homepage
- Navigation Bar
- Banner Section
- Contact Page
- Responsive User Interface


## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
